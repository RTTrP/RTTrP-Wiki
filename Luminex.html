Luminex Ethernet-DMX Nodes

The Luminex Implementation of RTTrPL

Information

Luminex Ethernet-DMX nodes receive RTTrPL to merge the data with sACN or ArtNet form any lighting control desk. The desk can control the merging by using a cross-fade trigger channel from the desk. All this being validated by Luminex and BlackTrax.

Luminex - RTTrPL

  • Ethernet-DMX nodes come default with an ArtNet IP address (2.x.y.z / 255.0.0.0). This has no impact on subscribing to the multicast RTTrPL addresses.
